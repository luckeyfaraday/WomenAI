# WomenAI Social Media Posts

This folder contains pre-written social media content for WomenAI's Twitter/X account.

## Folder Structure

- `launch/` - Launch announcement posts
- `features/` - Feature highlight posts
- `engagement/` - Community engagement posts
- `education/` - Health tips and educational content
- `support/` - Motivational and supportive messages

## Post Format

Each post file should include:
- Post text
- Hashtags
- Suggested media/images
- Best time to post
- Category/theme
